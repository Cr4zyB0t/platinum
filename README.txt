# INSTALLATION
    Pour install Platinum correctement, il sera obligation d'installer la version 3 de python ainsi que pip3.
    Vous pouver install python3 sur le site web officiel : https://www.python.org/

    # WINDOWS
        Sur windows vous pouvez lancer l'exécutable sans avoir besoin d'installer un logiciel tiers. 
        OU
        Vous pouvez aussi installer python3 puis lancer l'installateur via l'invite de commande avec les commandes suivantes : 

        // sudo python3 setup.py
        // python3 platinum.py

    # LINUX
        Lancer simplement le setup avec la commande suivante :
        // sudo python3 setup.py
        // python3 platinum.py
    
    # MAC OS
        Si vous voulez utiliser Metasploit, il faudra l'installer manuellement.
        Faite la commande suivante pour installer platinum : // sudo python setup.py
        Vous pouvez maintenant plancer platinum avec la commande suivante : // python3 platinum.py
